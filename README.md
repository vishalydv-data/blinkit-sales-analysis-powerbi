
# Blinkit Sales Analysis Dashboard

This project analyzes sales performance of Blinkit outlets using Power BI.  
The goal is to understand sales trends, product performance, outlet performance and customer ratings.

---

## Project Objective

To conduct a comprehensive analysis of Blinkit's sales performance, customer satisfaction and outlet distribution to identify insights and opportunities for business improvement.

---

## Tools Used

Power BI  
Excel  
DAX  
Data Cleaning  
Data Visualization

---

## Dataset Information

The dataset contains information about:

Item Type  
Fat Content  
Outlet Size  
Outlet Location  
Outlet Type  
Item Rating  
Item Visibility  
Sales

---

## KPIs (Key Performance Indicators)

Total Sales  
Average Sales  
Number of Items  
Average Rating

---

## Dashboard Insights

1. Sales by Fat Content
2. Sales by Item Type
3. Sales by Outlet Size
4. Sales by Outlet Location
5. Sales by Outlet Type
6. Outlet Establishment Trend

---

## Charts Used

Donut Chart  
Bar Chart  
Stacked Column Chart  
Line Chart  
Matrix Table

---

## Key Insights

Supermarket Type1 generates the highest sales.  
Tier 3 outlets contribute the highest revenue.  
Fruits and snack foods are among the highest selling item categories.  
Large outlets generate more revenue compared to small outlets.
